从Butterfly4.7.0开始，导航栏重写，不支持本教程的导航栏。
请尝试降级版本！

butterfly导航栏修改方案(自用方案) | 安知鱼 (anheyu.com)

自定义 css

添加自定义 css nav_menu.css, 注意其中的颜色代码需自行替换。这里提供一份我的颜色代码。

颜色 css

自定义 js

添加自定义 js，不会的同学参考Hexo 博客添加自定义 css 和 js 文件，开启 pjax，且引入的时候在标签上添加data-pjax属性。

main.js

为了处理顶栏一闪而过的 bug 修改本地themes/butterfly/source/js/main.js，注意有引用线上版本的话一定要使用本地版本改完再自行上传线上版本

header/index.pug

修改themes/butterfly/layout/includes/header/index.pug

其中nav-visible可以控制默认显示的是站点标题还是导航栏菜单。

nav.pug
